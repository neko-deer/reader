TUTORIAL:

make your own directory for manga/manhwa with this format:
[title]/
├── Chapter [num]/
│   ├── page_[num].[ext]

ex:
Iam gud nigga/
├── Chapter 1/
│   ├── page_01.jpeg

update titles.json with UPDATE.bat

specify the base directory containing the titles
base_dir = 'path'

START.bat to start reading
